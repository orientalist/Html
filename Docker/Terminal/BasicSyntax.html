Basic Syntax
Syntax Description
docker ps To show status of container.
docker images To show all local images.
docker pull [name of image]:[tags] To get specific image.
docker stop [first 4 characters of containerID] To stop specific ciontainer.
docker rm [first 4 characters of containerID] To remove specific container
docker rmi [first 4 characters of image] To remove specific image
docker run -it -p [local port]:[container port] -v [directory of local file]:[directory of container] To link files to specific container
docker run -p [local port]:[container port] [image]:[tags] [application of image] To run specific application of specific image.
docker start [first 4 characters of containerID.] To start specific sontainer.

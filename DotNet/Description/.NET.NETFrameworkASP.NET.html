.NET

微軟定義:
.NET是Microsoft為XML Web Service所提供的平台
這個平台有五大範圍:
  1. 工具(Tools)
  2. 伺服器(Servers)
  3. XML Web服務(XML Web Services)
  4. 用戶端(Clients)
  5. .NET操作環境(Experience)
簡言之.NET就是泛指與.NET程式開發相關的一切微軟產品與技術總合。

.NET Framework

簡言之:
.NET程式所需的環境，有了.NET Framework才能建置、部署及執行.NET應用程式。 重點:ASP.NET、C#、ADO.NET、LINQ

ASP.NET

ASP:Active Server Pages-動態伺服器網頁
基於.NET，依靠.NET Framework的函式庫用以開發網頁程式的"架構"， 可用多種語言(C#、VB)，可使用如Managed Common Language Runtime等.NET Framework的函式。

.NET Framework與ASP.NET的區別

  • .NET Framework:
    1. CLR(Common Lamguage Runtime):
      .NET程式的執行環境與引擎。
    2. Class Library(類別庫):
      現成的API，提供程式開發用，例如網頁需要"輸出檔案時"， 可在Server端直接呼叫I/O相關的類別。
  • ASP.NET
    是眾多.NET應用程式類型的一種，專注於網頁程式開發 ，會使用.NET Framework提供的類別庫。
    .NET Framework是所有應用程式共用的基礎設施。

Server vs Client端網頁技術

早期的網頁技術為純客戶端的Web技術，以HTML、JavaScript，CSS為主，但無法完成多項開發需求， 故出現將需求傳送到Server端，處理後再將結果以Render生成HTML、CSS、JavaScript的開發軟體。
ASP.NET即是其中之一，主體是伺服器技術，運用到:
  1. IIS(Internet Information Services)-網路資訊服務
  2. .Net Framework(提供程式基礎函式)
  3. ASP.NET(程式架構)

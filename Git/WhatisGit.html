What is Git?

簡介

Git是一種分散是版本控制器(Distributed Version Controll System)DVCS， 可將對檔案的每次"修改"紀錄為"版本"，以便在任何時候取得舊版的檔案。

分散式系統

相對於集中式版本控制系統(Centralize Version Controll System)如SVN、CVS， 其特點是需擁有一台專用伺服器，所有更新須與其溝通。
分散式版本控制器原則上有共同伺服器，但所有取得原始版本的個體擁有個人的完整版本， 可以在沒有網路的情況下正常運作。

Git 與 GitHub

Git是一款版本控制軟體，GitHub是一個商業網站，其本體為Git伺服器，可讓使用者 透過網頁操作Git。

與其他版控系統的差異

其他本控系統會記錄每個異動的片段內容，以拼湊各版本的完整檔案；而Git則以每個檔案 未修改(最新)為該版本該檔案(換句話說，如果A檔案經過10個版本都未變，那麼第10版的 A檔案會參考到第1版的A檔案)。

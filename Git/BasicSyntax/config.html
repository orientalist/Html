config

git config --list

顯示目前所有config設定

git config --global/--local user.name [username]

設定全域/本地的git使用者名稱

git config --global/--local user.email [user email]

設定全域/本地的git使用者信箱

git config --global core.editor [editor(emacs)]

更換全域的文字編輯器

git config --global alias.[縮寫] [command]

將全域command指派給特定的縮寫，如:
git config --global alias.st status
若指派的command帶有參數，需加入""，如:
git config --global alias.l "log --oneline --graph"

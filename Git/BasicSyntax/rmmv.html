rm mv

git rm [file]

直接將檔案刪除，並且將該修改加入暫存區。
原流程:
rm file→git add file→git commit -m ""
本流程:
git rm file→git commit -m ""

參數

  • git rm [file] --cached:
    非刪除!僅是解除git控管該檔案。

git mv [file] [new file name]

直接將該檔案改名並放入暫存區。
原流程:
mv file [new file name]→git add [new file name]→git commit -m ""
本流程:
git mv file [new file name]→git commit -m ""

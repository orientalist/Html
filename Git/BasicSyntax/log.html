git log

用以檢查Commit紀錄，越新的在越上面。
  • 黃字為sha1演算碼，代表該次commit的識別碼，不重複。
  • Author:commit者、信箱。
  • Date:commit時間
  • 下方為該次commit訊息。

參數

  • git log --oneline:
    僅顯示部分sha1碼以及commit訊息。
  • git log --graph:
    圖像化commit紀錄。
  • git log --author="user name":
    找尋特定使用者的commit，不須完整輸入。
    可搭配|(或)搜尋
    git log --author="aaa|bbb"
  • git log --grep="words"":
    找尋包含定訊息的commit。
  • git log -S "words":
    找尋commit內的檔案包含特定文字的commit。
  • git log --since="begin time" --until="end time" --after="specific time":
    找尋在某時間區間內的所有commit。
    --after表示某時間"後"
    例:git log --since="2018/10/25 0am" --intil="2018/10/31 0am" --after="2018/10/26"
    經典範例:
    git log --since="1pm" --until="6pm" --after="2018/10/01"
    2018/10/01後每天下午1到6點的commit

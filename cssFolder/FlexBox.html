Flex Box

模型概念

與一般盒子類模型差別在於:FlexBox具有 水平起始/終點(main start/main end)， 垂直起始終點(cross start/cross end)， 水平軸與垂直軸(main axis/cross axis) ，以及 水平尺寸與垂直尺寸(main size/cross size)。

屬性

  • display
  • flex-direction
  • justify-content
  • align-items
  • align-self
  • align-content
  • flex-wrap
  • order
  • flex

align-content

與align-item類似，但用於 多行內容的垂直對齊
  • flex-start:對齊最上方的cross start
  • flex-end:對齊最下方cross end
  • center:垂直置中
  • space-between:將第一行與最後一行對齊corss start與cross end
  • space-around:每行平均分配間距
  • stretch:預設值，內元素撐開

flex-wrap

規範子元素的"換行"屬性，否則display值為flex或inline-flex之子元素會彈性撐滿父元素
  • nowrap:預設值，單行
  • wrap:多行
  • wrap-reverse:多行且元素反轉

flex

flex由三個屬性組成，分別是:
  • flex-grow:數字，預設0，無單位，當該元素"小於"flex-basic值時，若為1則按比例"伸展"。
    flex:1 0 200px;表示若分配的寬度"小於"200px，延伸
  • flex-shrink:數字，預設0，無單位，當該元素"大於"flex-basic值時，若為1則按比例"壓縮"。
    flex:0 2 300px;表示若分配到的寬度"大於"300px，壓縮
  • flex-basic:數字+單位，預設值為0，若auto，則等同自身寬度，表示該元素的基本寬度，用以供flex-grow與flex-shrink參照。

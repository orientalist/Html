Flex Box

模型概念

與一般盒子類模型差別在於:FlexBox具有水平起始/終點(main start/main end)， 垂直起始終點(cross start/cross end)， 水平軸與垂直軸(main axis/cross axis) ，以及水平尺寸與垂直尺寸(main size/cross size)。

屬性

  • display
  • flex-direction
  • justify-content
  • align-items
  • align-self
  • align-content
  • flex-wrap
  • order
  • flex

Example

display

display:flex與display:inline-flex大約等於display:block 及display:inline-block，但其下子元素有更多彈性設定且其後方元素不自動換行。
